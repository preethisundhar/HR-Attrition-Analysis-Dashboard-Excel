# HR Attrition Analysis Dashboard (Excel)

## 📊 Project Overview
This project presents an **HR Attrition Analysis Dashboard** created in Microsoft Excel to analyze employee attrition trends and workforce insights. The dashboard helps HR teams understand patterns in employee turnover using interactive charts, KPIs, and slicers.

The analysis focuses on identifying factors that influence attrition such as **age group, gender, department, job role, and business travel**.

---

## 🎯 Objectives
- Analyze employee attrition patterns
- Identify departments with high attrition
- Understand attrition by age group and gender
- Provide interactive visualization for HR decision making

---

## 🛠 Tools Used
- **Microsoft Excel**
- **Power Query** – Data cleaning and transformation
- **Pivot Tables** – Data aggregation and analysis
- **Pivot Charts** – Visual representation
- **Slicers** – Interactive filtering
- **Dashboard Design** – KPI cards and charts

---

## 📂 Dataset
The dataset contains HR employee data including:
- Employee ID
- Age
- Gender
- Department
- Job Role
- Business Travel
- Attrition
- Work Distance
- Job Satisfaction
- Salary information

---

## 📈 Dashboard Features
The dashboard includes the following insights:

- **Total Employees KPI**
- **Total Attrition KPI**
- **Attrition Rate**
- **Attrition by Age Group**
- **Attrition by Gender**
- **Attrition by Department**
- **Attrition by Job Role**
- **Attrition by Business Travel**
- **Work Distance vs Attrition Analysis**

Interactive **slicers** allow filtering by:
- Department
- Gender
